# tiketsDocumentacion

La aplicacion consta de una main activity que podra recibir tikets de error con varios elementos en ellos, solo se mostraran los tickets atraves de el uso de fragments.
Los tickets se podran enviar y guardar una vez rellenados. La mainActivity sera la unica activity de la aplicacion. 
Tendra que haber 3 fragmentos:
-Editar tickets
-Mostrar tickets
-Ver tickets
